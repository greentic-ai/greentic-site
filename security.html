Security

Security is the feature, not a checkbox.

Threat model

Assume breach. Design for least privilege. Contain execution in WASM sandboxes with capability-based access.

Data handling

  • Encryption in transit (TLS 1.3) and at rest (AES‑256).
  • Secrets stored with envelope encryption and rotation schedules.
  • Tenant isolation at compute, data, and network layers.

Identity & access

OIDC/SSO, SCIM, RBAC/ABAC with scoped tokens. Signed webhooks (HMAC).

Compliance

SOC2 Type II (in progress), GDPR-ready. DPA available on request.
